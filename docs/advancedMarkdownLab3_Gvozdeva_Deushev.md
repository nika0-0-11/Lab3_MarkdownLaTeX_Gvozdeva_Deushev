### Alert-блок

>[!TIP] 
> **Совет по использованию**: Для экранизирования специальных символов в Markdown используйте обратный слэш `\`. Например, чтобы написать `*курсив*` без форматирования, напишите \*курсив\*
 
### Escape-символы 

* Заголовок без форматирования: `\# Это не заголовок`
* Звездочка как текст: `\*Просто звездочка\*`

### HTML в Markdown

```html
<p style="background-color: #f8f9fa; padding: 15px; border-radius: 5px;">
    Этот параграф оформлен с помощью <strong>HTML-тега</strong> внутри Markdown-документа.
</p>
```

[^1]: Синтаксис Markdown поддерживается на GitHub, GitLab, в большинстве редакторов кода и систем документирования
